PC動作診断 — 完全ローカル

バージョン 1.0.0

ダウンロード

Windows 用です。ZIP を解凍して PcTroubleNavi.exe を起動してください。インストール不要です。

ZIP をダウンロード

機能

  • 簡易検査 … CPU・メモリ・ストレージ・GPU・イベントログなどを数十秒で収集・判定
  • 詳細検査 … 各項目の負荷テスト（約20分〜）
  • カスタム検査 … 選択した項目だけ詳細検査
  • ネットワーク検査 … 接続・Ping・速度・DNS の確認
  • 設定 … 表示言語、ストレージ書込みテストの同意、更新確認

自分でできる対策（PC初心者向け）

検査で「注意」「警告」「危険」が出たときに、まず試せることをまとめています。順に確認してみてください。

※ ここに今後、対策記事を追加していきます。

  • 例）CPU・GPUが高温 … 通気口の掃除、冷却ファンの確認、負荷の高いソフトを減らす
  • 例）メモリ不足の兆候 … 使っていないアプリを閉じる、スタートアップを整理
  • 例）ストレージの空きが少ない … 不要なファイルの削除、ディスクのクリーンアップ
  • 例）イベントログにエラー … Windows Updateの実行、再起動、同じ操作を繰り返して再現するか確認

具体的な手順は随時このページに追記します。アプリの「検査結果」画面に表示されたリンクからいつでもこのページを開けます。

通信は更新確認時のみ。診断データはすべて端末内で処理されます。
